\# 电信管理系统 (Telecom System)



\## 项目简介

这是一个电信管理系统，用于管理电信业务相关功能。



\## 功能特性

\###企业用户

&nbsp;- 登录验证

&nbsp;- 用户管理

&nbsp;- 信息修改

&nbsp;-  流量峰期分析

\###普通用户

&nbsp;- 登录验证

&nbsp;- 套餐查看

&nbsp;- 套餐修改

&nbsp;- 充值

&nbsp;



\## 技术栈

\- HTML/CSS/JavaScript

\- Java



\## 安装和使用

启动 TelecomSystemApplication.java
